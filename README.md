# Streamlit 튜토리얼 😁

`Streamlit`을 활용하여 파이썬 코드로 빠르게 웹앱(WebApp)을 생성할 수 있습니다.
사용법도 굉장히 쉽고, 무료 호스팅 기능도 지원해주기 때문에, 데모 앱을 만들어 볼 때 굉장히 유용합니다.

**설치**
```python
pip install streamlit
```
